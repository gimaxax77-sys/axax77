사용 순서

0-1 Gemini API 키 입력 · 저장 필수 — 최초 1회만 하면 계속 저장돼요.
0-2 캐릭터 설정 시트 작성 필수 — 여기 적은 내용이 모든 프롬프트에 자동 반영돼요. 컨셉 리스트·프리셋·백업은 선택이에요.
1 플랫폼(미드저니/GPT) & 아이디어 입력 — 플랫폼 선택은 필수, 아이디어 입력은 선택(비워두면 캐릭터 단독 컷으로 생성).
2 표정 · 제스처 · 사물 세트 일괄 생성 선택 — 한 번에 여러 장 뽑고 싶을 때만 사용해요. 안 써도 1번만으로 단일 생성이 가능해요.
3 생성된 프롬프트 확인 · 복사 — 자동으로 표시돼요. 즐겨찾기·기록에서 예전 결과도 다시 찾을 수 있어요.
4 Etsy 상품 설명 생성 선택 — 실제 판매 등록할 때만 필요해요.

STEP 0-1Gemini API 키필수

키는 이 브라우저의 localStorage에만 저장되며, 서버로 전송되지 않습니다. · 무료로 키 발급: aistudio.google.com/apikey
이 파일을 나만 쓰는 게 아니라 다른 사람도 접근 가능한 곳에 올려둘 경우, Google AI Studio에서 이 키에 HTTP 리퍼러(웹사이트) 제한을 걸어두는 걸 권장해요.

프리셋 관리선택

캐릭터 시트 + 스타일 + 옵션 조합을 이름 붙여 저장해두면, 캐릭터가 여러 개거나 시즌 테마가 바뀔 때 매번 새로 세팅 안 해도 돼요.

데이터 백업 · 복원선택

모든 데이터는 이 브라우저에만 저장돼요. 캐시를 지우거나 기기를 바꾸면 사라지니, 종종 파일로 내보내두는 걸 추천해요. (API 키는 보안을 위해 내보내기에 포함되지 않아요) 내보내기 파일명에는 캐릭터 이름이 자동으로 들어가요.
모바일이나 파일을 직접 열어서 쓰는 환경(file://)에서는 브라우저 보안 정책상 폴더 선택이 지원되지 않아요 — 이 경우 위쪽 "내보내기" 버튼으로 평소처럼 다운로드해주세요.
